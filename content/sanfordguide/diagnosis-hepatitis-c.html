HCV Diagnosis

Who to Test ?

  • The US Preventive Services Task Force recommends screening for all adults aged 18 to 79 years (JAMA 2020;323: 970)

Timing

  • When tests turn positive after acute infection:
    • HCV RNA: ~ 2 weeks after initial infection
    • HCV Antibody: 6-12 weeks after infection
      • NOTE: HCV Antibody positivity can be significantly delayed in HIV+ patients with low CD4 counts and advanced HIV disease; therefore, HCV RNA should be checked when clinical suspicion of HCV infection is present and antibody test is negative.

Testing

  • Categories of HCV diagnostic tests and test type:
    • HCV Antibody Test: EIA assay
    • HCV Quantitative Viral Load (HCV RNA): PCR
    • HCV Genotype (1-6): Sequencing

Testing Algorithm

StepAction/Result
1
  • Perform HCV EIA (Reflex testing algorithm: If EIA positive, automatic HCV RNA test)
2
  • If HCV EIA negative: Evaluate ALT
    • If ALT normal: No evidence for HCV infection; repeat testing as clinically indicated
    • If elevated ALT: Perform Quantitative HCV RNA
      • If Quantitative HCV RNA negative: resolved HCV infection; repeat testing as clinically indicated
      • If positive: Active HCV; determine HCV Genotype and treat (see HCV treatment)
3
  • If HCV EIA positive: Perform Quantitative HCV RNA (reflex testing)
    • If negative: Wait 4-6 months, re-perform test
    • If positive: Chronic HCV; determine HCV Genotype
4
  • Evaluate for treatment based on HCV Genotype, although several DAA regimens are pan-genotypic; see HCV Treatment
